# reclip

Менеджер истории буфера обмена для Linux (GNOME/Wayland) — аналог `Win+V`.

Обычный буфер обмена помнит только последнее скопированное. `reclip` хранит
**историю** копирований и позволяет быстро вернуть любое из них по горячей клавише.

> ⚠️ Проект в стадии проектирования. Кода пока нет — идёт проработка решений.

## Задумка (кратко)
- **Язык:** Rust · **GUI:** GTK4
- **Устройство:** один бинарь `reclip` с под-командами — фоновый демон следит за
  буфером и копит историю; пикер по `Super+V` показывает историю и возвращает выбор.
- **Цель MVP:** простая и быстрая история **текста**, реально работающая на GNOME/Wayland.
